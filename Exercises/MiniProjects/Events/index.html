• OnMouseOver

    Mouse over on the button to change the background color

  • OnMouseOut

    Hover on the button and move the cursor away

  • OnClick

    The event gets triggered when a button is clicked

  • OnDoubleClick

    This event gets triggerd when the button is double clicked

  • OnFocus

    This event gets triggered when the text input is in focused state

  • OnBlur

    This event is triggered when you type something and click anywhere outside the text box (when the input looses the focus)
